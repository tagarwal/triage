Tree display of test runs

This panel is used for displaying test runs in tree form. Select a test run in the left panel to see it displayed here.

Tree Manipulation

  • click to expand tree nodes beneath this node.
  • click to collapse tree nodes beneath this node.
  • use the Expand all and Collapse all links to expand or collapse all nodes in the tree.

Information Provided

  • a test .xml or .suc file with passing tests. Hovering provide summary information on xml files.
  • a test .xml or .dif file with one or more failing tests. Hovering provides summary information on xml files.
  • displays more information when hovered over. In some cases it will show the host status taken at a point in time (from history log), or it may show some detail on test run results. Hovering provides summary information.
  • link to related information, such as a corresponding build file.
  • a diagnostic log warning entry.
    • Hovering provides an abbreviated version of the full entry.
  • a diagnostic log error entry. Hovering provides an abbreviated version of the full entry.
  • a link typically displays the content of a log file in the detail pane.
  • zooms out to the full log file where a record was taken from. This is placed in in the detail pane.
  • fulltext - shows the full content of the referred file when hovered over.
  • endtext - shows the last couple of lines of the referred file file when hovered over.
